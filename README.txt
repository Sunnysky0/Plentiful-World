Plentiful World Mod Framework
=============================

This is the basic directory structure for the Plentiful World mod.

Directories:
- common/: Defines, technologies, units, etc.
- events/: Event files.
- history/: History files for countries and states.
- interface/: GUI and graphics definitions.
- localisation/: Localization files (YAML).
- map/: Map files (provinces, terrain, etc.).
- gfx/: Graphics assets (flags, portraits, etc.).

To get started:
1. Add your country history files to history/countries/.
2. Add state history files to history/states/.
3. Define new countries in common/country_tags/.
4. Modify the map in map/ directory.

Good luck with your modding!
