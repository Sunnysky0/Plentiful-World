# Tech Tree Designs for Plentiful World Mod

This directory contains the Mermaid diagrams for the new technology trees designed for the Plentiful World mod.
The setting assumes 1936 starts with OTL 2026 level technology.

## Files Structure

- **[01_infantry.mmd](01_infantry.mmd)**: **COMPREHENSIVE INFANTRY SYSTEM**
    - **Infantry Weapons**: Assault Rifles -> Gauss -> Plasma -> Particle Beam.
    - **Equipment**: Exoskeletons -> Power Armor -> Phase Shields.
    - **Special Forces**: Spec Ops (Stealth), Marines (Amphibious Mecha), Paratroopers (ODST), Mountaineers.
    - **Mechanized & Motorized**: Trucks (Anti-gravity), IFVs, Amphibious Vehicles, Walker Mechs.
    - **Support & C4I**: Drones, Neural Network Comms, Maintenance, Military Police.
    - **Medical & Bio-Enhancement**: Stims, Berserk Drugs (An Lushan), Focus Pills (Imperial), Super Soldier Serum.

- **[02_support.mmd](02_support.mmd)**: **STRATEGIC SUPPORT**
    - Logistics (Orbital Drop), Medical (Regeneration), Recon (Satellites), Signal.

- **[03_armor.mmd](03_armor.mmd)**: **HEAVY ARMOR & MECHA**
    - Main Battle Tanks (Railgun/Plasma), Heavy Assault Platforms, Siege Mecha.

- **[04_artillery.mmd](04_artillery.mmd)**: **FIRE SUPPORT**
    - Towed, SPA, Rocket/Missile, Anti-Air, Desire Weapons (Pheromone Cannons).

- **[05_naval.mmd](05_naval.mmd)**: **NAVAL FORCES**
    - Screens, Capital Ships, Carriers, Submarines (Hypersonic).

- **[06_naval_support.mmd](06_naval_support.mmd)**: **NAVAL SYSTEMS**
    - Fire Control, Power Systems, Defense (Shields).

- **[07_air.mmd](07_air.mmd)**: **AIR FORCE**
    - Fighters (6th Gen+), CAS, Bombers (Sub-orbital), Transport, Special.

- **[08_industry.mmd](08_industry.mmd)**: **INDUSTRY & ECONOMY**
    - Manufacturing, Resource Extraction, Crystal/Desire Tech, Correction.

- **[09_advanced_tech.mmd](09_advanced_tech.mmd)**: **FUTURE TECH**
    - Computing, Nuclear/Physics, Biology, Space.

## How to View
You can view these diagrams using a Mermaid Live Editor or by installing a Mermaid extension for VS Code.
