# 中华帝国 (Imperial China) - CHI

## 1. 核心概念
**领袖**：**紫叶李 (Ziye Li)** —— 第十一皇女，帝国皇家科学院（负责晶体研究）的幕后掌控者，冷酷的理性主义者，被称为“晶体女皇”。
**风格**：赛博朋克风的封建帝国，结合了中国古代政治架构与近未来/黑科技设定（晶体能源）。宫殿由巨大的发光晶体柱支撑，禁卫军身穿动力甲。
**目标**：清洗腐败的“旧晶体官僚”，击败试图垄断能源的叛军，重塑帝国，建立绝对理性的“完美能源循环”秩序。

## 2. 故事线与国策树 (Focus Tree Overview)

### 起始阶段：危如累卵
*   **国策：帝国的黄昏**
    *   *效果*：获得大量Debuff（“宦官干政”、“晶体纯度下降”、“能源走私泛滥”）。
*   **事件：总理之死** -> 触发安禄山造反（指责中央克扣晶体） -> 触发东南互保（切断资金流）。
*   **国策：怀仁堂之变**
    *   *条件*：安禄山起兵后。
    *   *效果*：紫叶李发动政变上台，软禁老皇帝。清除“宦官干政”，获得新特质“铁血摄政王”（政治点数+20%，稳定度+10%）。解锁后续政治线。

### 政治路线 (Political Paths)

#### A. 绝对理性线 (The Path of Absolute Rationality) - 科学/飞升路线
*   **核心理念**：将国家重塑为高效的战争机器，消除人的“感性”弱点。人类存在的意义就是生产高质量的欲望结晶。
*   **关键人物**：商鞅（法家执行官/配额管理）、李时珍（生物科技）、鲁班（机械化）。
*   **国策**：
    *   **重组能源部**：建立以科学家和技术官僚为主的决策机构，取代旧贵族。
    *   **推行新连坐法**：商鞅主导，实行严格的“产出配额制”，完不成结晶上缴指标的村庄将受到惩罚。
    *   **晶体飞升 (Crystal Ascension)**：利用李时珍的生命科学技术，开启“人体强化计划”。
        *   **特质**：解锁“生化士兵”编制（高组织度，低人力消耗）。
    *   **机械化大军**：鲁班主导，快速工业化，生产“公输重工”系列晶体动力坦克。
    *   **终极国策：格式塔意识**：通过晶体共鸣网络，统一国民思想，消除叛乱，但也消除了个人意志。
*   **结局**：建立“机械飞升”的中华帝国，征服世界，将全人类纳入能源网络。

#### B. 皇道复兴线 (The Path of Imperial Restoration) - 传统/仁政路线
*   **核心理念**：恢复皇室威严，强调“君舟民水”，认为只有快乐的民众才能产出最高纯度的结晶。
*   **关键人物**：苏秦（外交/贸易）、郑和（海军/远征）、上官婉儿（内政/文化）。
*   **国策**：
    *   **万国来朝**：苏秦主导，开放晶体贸易，换取列强的租借法案和技术援助。
    *   **仁政与极乐**：废除残酷的配额制，建设更多的公共娱乐设施（“极乐园”），通过提升国民幸福感来自然增加结晶产量（高稳定度，高人力）。
    *   **远洋舰队计划**：郑和主导，重建以庞大“宝船”（晶体动力航母）为核心的帝国海军，目标是收回东南互保区的财富。
    *   **文化霸权**：向世界输出帝国的“房中术”文化，增加国际影响力。
*   **结局**：建立“大东亚共荣圈”式的朝贡体系，成为仁慈的霸主，垄断全球高端晶体市场。

### 军事路线 (Military Paths)

*   **京师卫戍战**：以袁崇焕为核心，依托**墨子**设计的“非攻力场发生器”（大型晶体护盾），死守直隶。
*   **闪电反击**：以霍去病、李广为核心，组建装备了“晶体助燃剂”的高机动部队，利用粉红闪电战突袭叛军后方。
*   **总体战**：以白起为核心，不惜一切代价（包括使用导致土地废弃的“枯萎弹”）歼灭敌人。

## 4. 特殊机制：宫廷与内阁 (Court & Cabinet)
*   **内阁平衡**：玩家需要在“法家”（商鞅/白起，注重效率/战争/强制生产）和“儒家/纵横家”（苏秦/上官婉儿，注重稳定/外交/自然生产）之间保持平衡。
*   **肃清度**：通过决议清洗旧贵族和宦官势力（魏忠贤残党），肃清度越高，行政效率越高，但可能引发叛乱。

## 5. 独特科技/兵种
*   **公输机关兽**（替代重型坦克）：鲁班设计的步行机甲，使用高能晶体核心，能发射激光。
*   **神机营**（特种部队）：装备晶体爆能枪的精锐步兵，对装甲单位有特效。
*   **楼船舰队**：复古风格的现代化战列舰，装备有巨大的“龙炎”喷火器。
*   **炼丹术**：一系列生物科技，可解锁“超级士兵血清”。
