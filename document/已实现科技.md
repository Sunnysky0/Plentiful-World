# Plentiful World 已实现科技（持续更新）

## 1. 步兵武器系统（已实现）

### 1.1 主武器链

- `pw_infantry_weapons0`（1936）：模块化突击步枪
- `pw_infantry_weapons1`（1936）：智能追踪子弹步枪
- `pw_infantry_weapons2`（1942）：便携式高斯步枪
- `pw_infantry_weapons3`（1946）：次世代高斯步枪

主链关系：

- `pw_infantry_weapons0 -> pw_infantry_weapons1 -> pw_infantry_weapons2 -> pw_infantry_weapons3`

### 1.2 支援链

- `pw_support_weapons1`（1936）：支援机枪
- `pw_support_weapons2`（1940）：班组情欲空气燃烧弹
- `pw_support_weapons3`（1943）：欲望过载媚药燃烧器

支援链关系：

- `pw_infantry_weapons1 -> pw_support_weapons1 -> pw_support_weapons2 -> pw_support_weapons3`

### 1.3 同代对齐规则（当前已采用）

- 支援机枪与追踪步枪同属 1936 年代，在科技树中按同一 `y` 对齐。
- 通过 `x` 偏移区分节点，避免卡片重叠。

## 2. 步兵装备解锁（已实现）

为避免国家专属装备命名覆盖科技显示，主武器链使用独立装备键：

- `pw_infantry_equipment_1`
- `pw_infantry_equipment_2`
- `pw_infantry_equipment_3`
- `pw_infantry_equipment_4`

对应原则：

- 科技用 `enable_equipments` 解锁上述 `pw_` 装备键。
- 图标映射使用 `GFX_pw_infantry_equipment_*_medium`。
- 本地化使用 `pw_infantry_equipment_*` 与 `_short/_desc`。

## 3. UI 显示约定（已实现）

- 装备科技定义包含 `show_equipment_icon = yes`。
- `technology_info_window` 中包含 `equipment_icon`（以及 `carrier_fighter_icon`）节点，保证详情页可显示装备图标。
